# 穿剑 (Chuan Jian) - 四人国风棋牌游戏

四人暗牌队友对战，红桃四叫牌找队友。国风水墨主题，逐张发牌动画，剑雷炸弹特效。

## 快速开始

```bash
cd server
npm install
npm start
```

访问 http://localhost:3000

## 游戏规则

### 牌具
- 52张标准扑克（无大小王）
- 点数大小：4 < 5 < 6 < 7 < 8 < 9 < 10 < J < Q < K < A < 2 < 3
- 花色不参与比大小

### 庄家与叫牌
- 红桃4持有者固定为庄家
- 庄家从自己没有的牌中叫一张（精确花色+点数）
- 持有该牌的玩家为秘密队友，叫牌打出前身份隐藏

### 牌型
- 基本：单张、对子、三不带、三带一、三带二
- 顺子：≥3张连续，范围4~A
- 连对：≥2组连续对，范围4~3（含A-2-3连接）
- 特殊：剑（44A）< 小雷（666）< 大雷（QQQ）< 炸弹（四同点）

### 特殊攻击牌型
| 牌型 | 压制 | 说明 |
|------|------|------|
| 剑·44A | 压所有基本/顺子/连对 | 同级不可互压 |
| 小雷·666 | 压剑及所有普通牌 | |
| 大雷·QQQ | 压小雷、剑 | |
| 炸弹 | 压一切，比点数 | 每颗炸弹最终积分×2 |

### 积分
- 大赢（1、2名）：每人+2
- 小赢（1、3名）：每人+1
- 平局（1、4名）：0分
- 炸弹翻倍：每出一次炸弹，最终积分×2

## 技术架构

- 前端：HTML/CSS/JS + Socket.IO 国风UI
- 后端：Express + Socket.IO + Node.js
- 国风设计：水墨宣纸底纹、书法字体、中式牌面
- AI 难度：简单（随机） / 中等（策略保留、配合推断）

## 特效（游戏中触发）

- **发牌**：牌从中心逐张飞向四家（0.15s/张），最后自动理牌
- **剑**：三牌发光凝成古剑虚影穿刺飞出
- **小雷**：牌面闪烁雷纹，落雷炸开
- **大雷**：雷云旋转三道粗雷落下，震屏
- **炸弹**：牌放大爆炸火光扩散，碎片飞溅
- **队友揭示**：头像闪烁金边，盖"剑盟"印章
